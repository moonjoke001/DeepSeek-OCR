# Ubuntu系统截图方法 1 2 3

text[[110, 355, 485, 395]]
在Ubuntu系统中，您可以通过多种方式进行截图，包括快捷键、系统自带工具和第三方软件。以下是详细介绍：

title[[110, 411, 191, 428]]
## 方法一：使用快捷键

text[[111, 442, 485, 483]]
1．全屏截图 按下PrintScreen键（部分键盘标为PrtSc），即可截取整个屏幕，截图会自动保存到图片文件夹。

text[[111, 500, 357, 518]]
2．当前窗口截图 按下Alt＋PrintScreen，仅截取当前活动窗口。

text[[111, 533, 392, 551]]
3．区域截图 按下Shift＋PrintScreen，然后用鼠标选择需要截取的区域。

text[[111, 566, 485, 607]]
4．保存到剪贴板 全屏：按Ctrl＋PrintScreen。当前窗口：按Ctrl＋Alt＋PrintScreen。区域：按Shift＋Ctrl＋PrintScreen。

title[[110, 631, 234, 649]]
## 方法二：使用系统自带截图工具

text[[111, 664, 280, 682]]
1．打开应用菜单，搜索并启动“截图”工具。

text[[111, 697, 485, 738]]
2．在工具中可以选择以下功能：截取全屏、当前窗口或自定义区域。设置延时截图（适用于截取菜单项）。是否包含鼠标指针。

title[[110, 763, 210, 781]]
## 方法三：使用第三方软件

text[[111, 796, 485, 835]]
·Flameshot 安装命令：sudo apt install flamesthot 使用快捷键或命令行启动，支持标注和编辑功能。
